## 📋 Descripción

Aplicación web progresiva (PWA) para clientes de Piums Platform. Permite a los usuarios buscar artistas, hacer reservas, gestionar pagos y comunicarse con artistas en tiempo real.

**Dominio**: `piums.com` | **Puerto dev**: `3000`

## ✨ Características

- 🔍 Búsqueda avanzada de artistas
- 📅 Sistema de reservas con calendario interactivo
- 💳 Pagos seguros con Stripe
- ⭐ Reseñas y ratings
- 💬 Chat en tiempo real
- 📊 Dashboard personal
- 📱 PWA instalable

## 🚀 Inicio Rápido

```bash
cd apps/web-client/web
pnpm install
cp .env.example .env.local
pnpm dev  # http://localhost:3000
```
